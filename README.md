# Rook AI — Expo (Replit + Expo Go)

This is a minimal managed-Expo skeleton configured to run on Replit and connect to Expo Go.

Quick start (Replit):
1. Create a new Replit and import this GitHub repository.
2. Go to the Secrets panel and add any API keys (e.g. GEMINI_API_KEY) if you need them.
3. Click Run — Replit will install dependencies and start Expo (tunnel).
4. Open the Expo Dev Tools link shown in the Replit console, then scan the QR with Expo Go.

Notes:
- This repo contains placeholder modules (src/modules/*). Integrate your model API and implement the Rook Personality rules in those modules.
- Replace assets/icon.png and assets/splash.png with real images for a polished experience.
